# 📋 MetaStock 產品規劃

> 建立日期：2026/01/22

---

## 產品定位

### 核心定位：選股策略工具

**不是**做一個完整的股票資訊平台（如 Yahoo 股市、CMoney）  
**而是**專注於「選股策略」這個垂直領域

| 我們做 ✅ | 我們不做 ❌ |
|----------|------------|
| 題材追漲篩選器 | 大盤總覽頁 |
| 箱型突破篩選器 | 產業分類頁 |
| 短線飆股雷達 | 新聞整合 |
| 個股簡易資訊（支援策略）| 完整財報分析 |

### 為什麼這樣定位

| 比較 | 完整股票平台 | 選股策略工具 |
|------|-------------|-------------|
| 開發時間 | 3-6 個月 | 1-2 個月 |
| 競爭對手 | Yahoo、CMoney、Goodinfo | 幾乎沒有 |
| 差異化 | 低 | **高** |
| 資源集中 | 分散 | **專注** |

## 資料規劃

### 後端必須撈取的資料 ✅

這些是策略計算必須的，不管有沒有 UI 都要撈：

| 資料類型 | 用途 | 來源 |
|----------|------|------|
| 每日股價（開高低收量）| 計算箱型、突破、K線 | 證交所 |
| 三大法人買賣超 | 計算籌碼評分 | 證交所 |
| 近 20/60 日歷史價格 | 計算均線、支撐壓力 | 資料庫 |
| 集保戶股權分散表 | 大戶持股變化（選用）| 證交所（週報）|

### 前端呈現 vs 不呈現

| 資料 | 策略計算 | UI 呈現 |
|------|:--------:|:-------:|
| 個股股價 | ✅ | ✅ 簡易個股頁 |
| K 線圖 | ✅ | ✅ 簡易個股頁 |
| 法人買賣超 | ✅ | ✅ 簡易個股頁 |
| 大盤指數 | ❌ | ❌ 不做 |
| 產業分類 | ❌ | ❌ 不做 |
| 新聞 | ❌ | ❌ 不做 |

## UI/導航規劃

### 導航架構（混合式）

**頂部導航**：精簡，只放核心入口
```
MetaStock | 首頁 | 個股查詢 | 🔍搜尋
```

**首頁**：功能卡片牆，展示所有策略
```
┌─ 選股策略 ──────────────────────────────┐
│  [題材追漲] [箱型突破] [短線飆股] [更多…] │
└─────────────────────────────────────────┘

┌─ 分析工具 ──────────────────────────────┐
│  [K線分析] [籌碼分析] [法人動向]          │
└─────────────────────────────────────────┘
```

### 頁面結構

```
MetaStock
├── 首頁（功能入口）
│   ├── 選股策略卡片區
│   ├── 分析工具卡片區
│   └── 快速搜尋
│
├── 選股策略（核心功能）
│   ├── 題材追漲分析器
│   ├── 箱型突破篩選器
│   └── 短線飆股雷達
│
├── 個股查詢（輔助功能）
│   ├── 搜尋股票
│   └── 個股詳情頁（股價、K線、法人）← 已完成
│
└── 我的帳戶（未來）
    ├── 持股管理
    └── 交易紀錄
```
