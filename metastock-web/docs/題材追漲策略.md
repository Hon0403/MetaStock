## 策略總覽

### 評分系統（滿分 10 分）

| 構面 | 權重 | 滿分 | 子項目 |
|:----:|:----:|:----:|--------|
| **題材面** | 40% | 4.0 | 強度(1.5) + 持續性(1.0) + 新鮮度(1.0) + 真實性(0.5) |
| **籌碼面** | 40% | 4.0 | 法人買超(2.0) + 連續性(1.0) + 成交量(0.5) + 大戶持股(0.5) |
| **追漲安全** | 20% | 2.0 | 追漲階段(1.5) + 賠率(0.5) |

```

```

## Step 2：題材查證

> 目的：避免追假消息、避免被騙線

### 查證流程

| 步驟 | 內容 | 標準 |
|:----:|------|------|
| 1 | **確認消息源頭** | 第一手(官方)✅ / 第二手(媒體)⚠️ / 第三手(傳聞)❌ |
| 2 | **查證官方網站** | 政府官網、證交所、公司官網 |
| 3 | **查證主流媒體** | 至少3家媒體報導且內容一致 |
| 4 | **比對時間差** | <7天✅ / 7-14天⚠️ / >14天❌ |
| 5 | **驗證真實性** | 高(官方確認) / 中(公司聲明) / 低(傳聞) |

### 可信度等級

| 等級 | 消息來源 | 舉例 | 決策 |
|:----:|----------|------|------|
| **高** | 官方確認 | 政府公告、公司公告 | ✅ 可以找股票 |
| **中** | 公司聲明 | 董事長說、尾牙透露 | ⚠️ 謹慎評估 |
| **低** | 市場傳聞 | 網路消息、LINE群組 | ❌ 不建議操作 |

## Step 3：三大核心評估

### 3.1 題材面評估 (40%)

#### 評分項目

| 項目 | 配分 | 評分標準 |
|------|:----:|----------|
| **題材強度** | 1.5 | 根據受惠比例：<10%(0.3) / 10-20%(0.6) / 20-40%(0.9) / 40-70%(1.2) / >70%(1.5) |
| **題材持續性** | 1.0 | 短線3-7天(0.3) / 中線1-2月(0.7) / 長線>3月(1.0) |
| **題材新鮮度** | 1.0 | ≤3天(1.0) / 4-7天(0.7) / 8-14天(0.4) / >14天(0) |
| **真實性** | 0.5 | 已確認(0.5) / 部分確認(0.3) / 未確認(0) |

#### 評級標準

| 總分 | 評級 |
|:----:|:----:|
| ≥3.5 | 優秀 ✅✅✅ |
| 3.0-3.4 | 良好 ✅ |
| 2.5-2.9 | 及格 ⚠️ |
| <2.5 | 不及格 ❌ |

---

### 3.2 籌碼面評估 (40%)

> 核心理念：法人連買才是真買，散戶追高要小心

#### 評分項目

| 項目 | 配分 | 評分標準 |
|------|:----:|----------|
| **法人買超** | 2.0 | 單日(1.0) + 累計5日(1.0)，加權：外資50%/投信35%/自營15% |
| **連續性** | 1.0 | ≥5天(1.0) / 3-4天(0.7) / 2天(0.5) / 1天(0.3) / 轉賣超(0) |
| **成交量** | 0.5 | 對比20日均量：≥3倍(0.5) / 2-3倍(0.4) / 1.5-2倍(0.3) |
| **大戶持股** | 0.5 | 週增幅：≥3%(0.5) / 2-3%(0.4) / 1-2%(0.3) |

#### 法人買超細項

| 級距 | 單日評分 | 累計5日評分 |
|------|:--------:|:----------:|
| ≥500張 | 1.0 | - |
| 300-499張 | 0.7 | - |
| 100-299張 | 0.4 | - |
| <100張 | 0.2 | - |
| ≥1000張 | - | 1.0 |
| 500-999張 | - | 0.7 |
| <500張 | - | 0.4 |

### 3.3 追漲安全性評估 (20%)

> 核心理念：只追初升段，不追主升段，絕不追末升段

#### 追漲階段定義

| 階段 | 漲幅 | 評分 | 建議部位 | 停損 | 目標 |
|:----:|:----:|:----:|:--------:|:----:|:----:|
| **初升段** | 0-12% | 1.5 | 15-40% | -5.5% | +12-15% |
| **主升段** | 12-25% | 0.2-0.5 | 7.5-15% | -6% | +10% |
| **末升段** | >25% | 0 | ❌不做 | - | - |

#### 賠率計算

```
賠率 = (目標價 - 買進價) / (買進價 - 停損價)
```

| 賠率 | 評分 |
|:----:|:----:|
| ≥2.5:1 | 0.5 ✅✅✅ |
| 2.0-2.4:1 | 0.4 ✅✅ |
| 1.5-1.9:1 | 0.3 ✅ |
| <1.5:1 | 0 ❌ |

## Step 4：風險控管檢查

> 必須全數通過才能進場

### 9項檢查清單

| # | 項目 | 條件 | 不通過處理 |
|:-:|------|------|-----------|
| 1 | 題材已查證 | 可信度 ≥ 中等 | ❌ 直接放棄 |
| 2 | 籌碼連續轉強 | 法人連買 ≥ 2日 | ⚠️ 等待確認 |
| 3 | 追漲階段符合 | 漲幅 ≤ 12% | ❌ 不追高 |
| 4 | 賠率合理 | 賠率 ≥ 1.5:1 | ❌ 風險太大 |
| 5 | 週交易次數 | 本週 < 5次 | ⚠️ 過度交易 |
| 6 | 無連虧或週虧 | 連虧<2次 且 週虧<8% | ❌ 暫停交易 |
| 7 | 盤中非急拉 | 30分鐘內漲幅 < 5% | ⚠️ 可能誘多 |
| 8 | 非情緒化決定 | 符合選股邏輯才買 | ⚠️ 冷靜一天 |
| 9 | 非處置股 | 成交量 > 1000張 | ❌ 難以出場 |

### 判定標準

| 通過項數 | 決策 |
|:--------:|------|
| 9/9 | ✅ 可以進場 |
| 7-8/9 | ⚠️ 謹慎考慮 |
| <7/9 | ❌ 不建議進場 |

## Step 5：最終排序與選擇

### 計算公式

```
總分 = 題材面(4.0) + 籌碼面(4.0) + 追漲安全(2.0) = 10.0
```

### 排序標準

1. **第一順位**：總分高低
2. **第二順位**：題材新鮮度
3. **第三順位**：籌碼連續性

### 篩選條件

| 總分 | 決策 |
|:----:|------|
| ≥8.5 | ✅ 優先考慮 |
| 8.0-8.4 | ⚠️ 可以考慮 |
| <8.0 | ❌ 不建議 |
