## 策略總覽

### 策略參數

| 項目 | 數值 |
|:----:|:----:|
| 持有週期 | 5-10 天 |
| 目標報酬 | +5-8% |
| 停損設定 | -2% 至 -3% |
| 勝率目標 | 60-65% |

### 四大關鍵

1. **整理辨識**：找出箱型整理的股票
2. **突破確認**：確認真突破（非假突破）
3. **進場時機**：突破當天或隔日
4. **出場紀律**：達標出場、破線停損

## Step 1：箱型整理辨識

### 什麼是箱型整理

```
高點 55 ────────────────── 壓力線
         │震盪│震盪│震盪│
低點 50 ────────────────── 支撐線
```

### 辨識條件

#### 必要條件（全部符合）

| 條件 | 標準 |
|------|------|
| 近10日漲跌幅 | ±8% 內（橫盤） |
| 最高與最低價差距 | 5-10%（箱子高度） |
| 整理時間 | ≥10 天 |
| 成交量 | 逐步萎縮 |

#### 加分條件

| 條件 | 說明 |
|------|------|
| 均線糾結 | 5/20/60日線距離 < 5% |
| 明顯高低點 | 至少碰 3 次 |
| 多頭整理 | 股價在均線之上 |
| 前有漲幅 | 整理後再起 |

### 整理期間評分

| 天數 | 評分 | 說明 |
|:----:|:----:|------|
| 10-15 天 | 0.5 | 剛開始整理 |
| 15-20 天 | 1.0 ✅ | 標準整理 |
| 20-30 天 | 0.7 | 整理較久 |
| >30 天 | 0.3 | 可能無效 |

## Step 2：突破訊號確認

### 真突破 vs 假突破

| 類型 | 特徵 |
|:----:|------|
| **真突破** | 帶量突破、收盤站穩壓力線上 |
| **假突破** | 量縮、尾盤拉抬、長上影線 |

### 真突破的 5 大特徵

| # | 特徵 | 配分 | 標準 |
|:-:|------|:----:|------|
| 1 | **量能爆大** ⭐⭐⭐⭐⭐ | 1.0 | ≥20日均量×2倍 |
| 2 | **K線強勢** ⭐⭐⭐⭐ | 1.0 | 紅K實體≥3%、上影線<1% |
| 3 | **突破時間** ⭐⭐⭐ | 1.0 | 早盤突破最佳 |
| 4 | **均線配合** ⭐⭐⭐ | 1.0 | 同步突破5/20日線 |
| 5 | **籌碼配合** ⭐⭐⭐ | 1.0 | 外資買超>100張 |

#### 量能評分細項

| 成交量倍數 | 評分 | 說明 |
|:----------:|:----:|------|
| ≥3倍 | 1.0 | 爆量突破 ✅✅✅ |
| 2-3倍 | 0.7 | 量能足夠 ✅✅ |
| 1.5-2倍 | 0.4 | 量能普通 ✅ |
| <1.5倍 | 0 | 量能不足 ❌ |

### 假突破特徵（要避開）

| 類型 | 特徵 | 結果 |
|------|------|------|
| 量縮突破 | 成交量<20日均量 | 1-2天縮回箱內 |
| 尾盤拉抬 | 最後30分鐘突然拉高 | 隔日開低 |
| 長上影線 | 上影線>3% | 壓力太重 |
| 高開低走 | 開盤跳空後一路下殺 | 出貨訊號 |

### 突破評級標準

| 總分 | 評級 | 建議 |
|:----:|:----:|------|
| ≥4.0 | ✅✅✅ | 真突破機率極高 |
| 3.0-3.9 | ✅✅ | 真突破機率高 |
| 2.0-2.9 | ⚠️ | 需要觀察 |
| <2.0 | ❌ | 可能假突破 |

## Step 3：進場時機與價位

### 三種進場時機

| 時機 | 優點 | 缺點 | 適合 |
|------|------|------|------|
| **突破當天收盤前** | 最快、不錯過 | 風險稍高 | 評分≥4.0 |
| **隔日開盤** | 已確認站穩 | 可能買貴 | 評分3.0-4.0 |
| **等拉回測試箱頂** | 風險最小 | 可能錯過 | 不急進場 |

### 進場檢查清單

- [ ] 箱型整理明確（10-20 天）
- [ ] 突破分數 ≥ 3.0
- [ ] 成交量放大 ≥ 1.5 倍
- [ ] 收盤站穩箱頂之上
- [ ] 均線開始向上
- [ ] 不是處置股
- [ ] 當日漲幅 < 7%
- [ ] 本週交易次數 < 5 次
- [ ] 符合資金配置

### 停損停利設定

| 類型 | 方法 | 公式 |
|------|------|------|
| **停損** | 跌破箱頂 | 箱頂 × 0.97 (-3%) |
| **停損** | 固定比例 | 買進價 × 0.975 (-2.5%) |
| **停利** | 保守目標 | 買進價 × 1.05 (+5%) |
| **停利** | 標準目標 | 買進價 × 1.08 (+8%) |

### 資金配置

| 類型 | 單筆比例 | 最多持有 | 現金保留 |
|------|:--------:|:--------:|:--------:|
| 保守型（新手）| 15-20% | 2 檔 | ≥50% |
| 標準型 | 20-30% | 3 檔 | ≥30% |
| 積極型 | 30-40% | 4 檔 | ≥20% |

## Step 4：持有管理與出場

### 每日檢查

| 時間 | 檢查項目 |
|------|----------|
| **早上 08:50** | 盤前新聞、國際股市、股票異常 |
| **晚上 20:30** | 法人買賣超、K線走勢、報酬計算 |

### 五種出場情境

| # | 情境 | 條件 | 動作 |
|:-:|------|------|------|
| 1 | **達標獲利** ✅ | 獲利 ≥ 5-8% | 賣出80%、保留20% |
| 2 | **碰壓力區** ⚠️ | 遇壓力開始回檔 | 先賣50% |
| 3 | **籌碼轉弱** ⚠️ | 法人連賣2天 | 出場觀望 |
| 4 | **破線停損** ❌ | 跌破停損價 | 立刻賣出 |
| 5 | **時間停損** ⚠️ | 持有10天未達標 | 出場換股 |

### 出場決策流程

```
持有中 → 每日檢查
    ├→ 獲利 ≥ 5%？     → 是 → ✅ 賣出
    ├→ 跌破停損？      → 是 → ❌ 立刻賣出
    ├→ 籌碼轉弱？      → 是 → ⚠️ 賣出觀望
    ├→ 碰壓力區？      → 是 → ⚠️ 先賣50%
    └→ 持有 ≥ 10天？   → 是 → ⚠️ 出場換股
```

### 常見錯誤

| 錯誤 | 狀況 | 教訓 |
|------|------|------|
| **捨不得賣** | 到目標想等更高 | 設定目標就要遵守 |
| **不願停損** | 想等反彈再賣 | 停損要果斷 |
| **追漲停** | 衝動追買漲停板 | 追在初升段(0-12%) |
| **頻繁換股** | 2天沒漲就換 | 給策略5-10天時間 |
| **滿倉操作** | 看到機會就全買 | 保留30%現金 |

## 附錄：Python 程式碼範例

```python
def calculate_breakout_score(stock, day):
    """計算突破分數（總分 5.0）"""
    
    # 1. 量能（1.0 分）
    volume_ratio = stock.volume[day] / avg(stock.volumes[-20:])
    volume_score = 1.0 if volume_ratio >= 3 else (0.7 if volume_ratio >= 2 else 0.4)
    
    # 2. K線強度（1.0 分）
    body = (stock.close[day] - stock.open[day]) / stock.open[day] * 100
    candle_score = 1.0 if body >= 3 else 0.6
    
    # 3. 均線配合（1.0 分）
    above_ma = stock.close[day] > stock.ma5[day] and stock.close[day] > stock.ma20[day]
    ma_score = 1.0 if above_ma else 0.3
    
    # 4. 籌碼配合（1.0 分）
    chip_score = 1.0 if stock.foreign_buy[day] > 100 else 0.3
    
    return volume_score + candle_score + ma_score + chip_score
```
