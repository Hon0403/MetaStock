Step 2：題材真實性查證
text
目的：
避免追假消息、避免被騙線

查證流程（5步驟）：

【2.1 確認消息源頭】
問題：
- 誰說的？（政府/公司/媒體/網友）
- 從哪裡來？（官網/記者會/傳聞）
- 是第一手還是轉述？

分類：
- 第一手：官方公告、記者會 ✅
- 第二手：媒體報導 ⚠️
- 第三手：網路傳聞 ❌

【2.2 查證官方網站】
必查清單：
- 政府官網（行政院、經濟部）
- 證交所重大訊息
- 公司官網公告
- 美國政府官網（政策類）

【2.3 查證主流媒體】
必查媒體：
- 國際：Bloomberg、Reuters、CNBC
- 台灣：經濟日報、工商時報、MoneyDJ

交叉比對：
- 至少 3 家媒體報導 ✅
- 報導內容一致 ✅
- 有官方引述 ✅

【2.4 比對時間差】
計算：發布日 → 今天 = ? 天

標準：
- < 7 天：新鮮 ✅✅✅
- 7-14 天：尚可 ⚠️
- > 14 天：老化 ❌

【2.5 驗證真實性】
評估標準：

Step 3：三大核心評估
3.1 題材面評估（40%）
text
評分項目（總分 4.0）：

【3.1.1 題材強度】（15% = 1.5分）
評估：這個題材對股票的影響有多大？

評分標準：
⭐ 1星（0.3分）：影響很小
  - 間接受惠
  - 影響比例 < 10%
  - 例：某客戶占比 5%

⭐⭐ 2星（0.6分）：影響小
  - 部分受惠
  - 影響比例 10-20%
  - 例：新產品占比 15%

⭐⭐⭐ 3星（0.9分）：影響中等
  - 直接受惠
  - 影響比例 20-40%
  - 例：訂單占比 30%

⭐⭐⭐⭐ 4星（1.2分）：影響大
  - 主要受惠
  - 影響比例 40-70%
  - 例：美國市場占比 60%

⭐⭐⭐⭐⭐ 5星（1.5分）：影響極大
  - 最大受惠
  - 影響比例 > 70%
  - 例：美國市場占比 80-90%

計算公式：
題材強度分數 = (星等 / 5) × 1.5


【3.1.2 題材持續性】（10% = 1.0分）
評估：這個題材會影響多久？

分類標準：
短線（0.3分）：3-7 天
  - 單一事件（記者會、新聞）
  - 快速炒作、快速退燒
  - 例：尾牙透露訂單

中線（0.7分）：1-2 個月
  - 政策效應（關稅、補助）
  - 產業趨勢（轉單效應）
  - 例：台美關稅協議

長線（1.0分）：3 個月以上
  - 結構改變（供應鏈重組）
  - 產業升級（AI、5G）
  - 例：AI 長期趨勢

計算公式：
題材持續性分數 = 
  - 短線：0.3
  - 中線：0.7
  - 長線：1.0


【3.1.3 題材新鮮度】（10% = 1.0分）
評估：這個題材公布多久了？

計算：發布日期 → 今天 = X 天

評分標準：
≤ 3 天（1.0分）：最新鮮 ✅✅✅
  - 剛公布
  - 市場還沒反應完
  - 追價空間大

4-7 天（0.7分）：新鮮 ✅
  - 開始發酵
  - 市場正在反應
  - 還有空間

8-14 天（0.4分）：開始老化 ⚠️
  - 已經炒過
  - 大部分人知道
  - 空間有限

> 14 天（0分）：已老化 ❌
  - 舊聞
  - 可能反應完畢
  - 不建議追

計算公式：
新鮮度分數 = 
  if 天數 ≤ 3: 1.0
  elif 天數 ≤ 7: 0.7
  elif 天數 ≤ 14: 0.4
  else: 0.0


【3.1.4 真實性】（5% = 0.5分）
評估：這個題材可信度如何？

評分標準：
已確認（0.5分）：✅
  - 官方公告
  - 政府公布
  - 公司正式聲明

部分確認（0.3分）：⚠️
  - 公司透露但未正式公告
  - 媒體報導但未官方確認
  - 董事長、高層說法

未確認（0分）：❌
  - 網路傳聞
  - 小道消息
  - 無法查證來源

計算公式：
真實性分數 = 
  if 已確認: 0.5
  elif 部分確認: 0.3
  else: 0.0


【題材面總分計算】
總分 = 題材強度 + 題材持續性 + 題材新鮮度 + 真實性
滿分 = 1.5 + 1.0 + 1.0 + 0.5 = 4.0

評級標準：
≥ 3.5：優秀 ✅✅✅
3.0-3.4：良好 ✅
2.5-2.9：及格 ⚠️
< 2.5：不及格 ❌

text
評分項目（總分 4.0）：

核心理念：
「法人連買才是真買，散戶追高要小心」

【3.2.1 法人買超】（20% = 2.0分）
評估：三大法人的買賣動作

資料來源：
- 證交所「三大法人買賣超日報」
- API: InstitutionalInvestorsBuySell

評分標準：

單日買超評分（1.0分）：
買超 ≥ 500張：1.0分 ✅✅✅
買超 300-499張：0.7分 ✅✅
買超 100-299張：0.4分 ✅
買超 < 100張：0.2分 ⚠️
賣超：0分 ❌

累計買超評分（1.0分）：
近5日累計 ≥ 1000張：1.0分 ✅✅✅
近5日累計 500-999張：0.7分 ✅✅
近5日累計 < 500張：0.4分 ✅
賣超：0分 ❌

計算公式：
法人買超分數 = min(單日評分 + 累計評分, 2.0)

細節權重：
外資買超權重：50%（最重要）
投信買超權重：35%（次重要）
自營商權重：15%（參考）

加權計算：
法人買超強度 = 
  (外資買超 × 0.5) + 
  (投信買超 × 0.35) + 
  (自營買超 × 0.15)


【3.2.2 連續性】（10% = 1.0分）
評估：法人是否持續買進

計算：連續買超天數

評分標準：
連續 ≥ 5天：1.0分 ✅✅✅
連續 3-4天：0.7分 ✅✅
連續 2天：0.5分 ✅
僅1天：0.3分 ⚠️
轉賣超：0分 ❌

判斷邏輯：
if 今天買超 and 昨天買超:
    連續天數 += 1
else:
    連續天數 = 0

計算公式：
連續性分數 = 
  if 連續天數 ≥ 5: 1.0
  elif 連續天數 ≥ 3: 0.7
  elif 連續天數 == 2: 0.5
  elif 連續天數 == 1: 0.3
  else: 0.0


【3.2.3 成交量】（5% = 0.5分）

放大 ≥ 3倍：0.5分 ✅✅✅（爆量）
放大 2-3倍：0.4分 ✅✅
放大 1.5-2倍：0.3分 ✅
放大 < 1.5倍：0.1分 ⚠️

成交量分數 =

  elif 倍數 ≥ 2: 0.4
  elif 倍數 ≥ 1.5: 0.3
  elif 倍數 ≥ 1: 0.1
  else: 0.0


【3.2.4 大戶持股變化】（5% = 0.5分）

- 持股 ≥ 400張的投資人

評分標準：
增加 ≥ 3%：0.5分 ✅✅✅
增加 2-3%：0.4分 ✅✅
增加 1-2%：0.3分 ✅
增加 < 1%：0.1分 ⚠️
減少：0分 ❌

計算公式：
大戶持股分數 = 
  if 變化 ≥ 3: 0.5
  elif 變化 ≥ 2: 0.4
  elif 變化 ≥ 1: 0.3
  elif 變化 > 0: 0.1
  else: 0.0


【額外指標（加分項）】

內外盤比例：

3.3 追漲安全性評估（20%）
text
評分項目（總分 2.0）：

核心理念：
「只追初升段，不追主升段，絕不追末升段」

【3.3.1 追漲階段判斷】（15% = 1.5分）
評估：股票目前在哪個階段

階段定義：

初升段（0-12%）：✅✅✅ 最佳進場區
特徵：
- 剛突破整理
- 剛開始上漲
- 還沒被大量關注
- 追價空間大

主升段（12-25%）：⚠️ 謹慎區
特徵：
- 已經漲一段
- 開始被關注
- 追價風險增加
- 可能隨時拉回

末升段（>25%）：❌ 禁止追高區
特徵：
- 已經漲很多
- 市場熱度高
- 追高風險極大
- 很可能是最後一棒

評分標準：
漲幅 0-12%：1.5分 ✅✅✅
漲幅 12-20%：0.5分 ⚠️
漲幅 20-25%：0.2分 ⚠️
漲幅 > 25%：0分 ❌


【計算方式】
關鍵：找到「起漲點」

起漲點定義：
- 突破前的整理低點
- 近期波段低點
- 題材發布前的價格

步驟：
1. 找出近 20 日最低點（短期低點）
2. 找出近 60 日最低點（中期低點）
3. 找出題材發布日的價格
4. 取三者中「最合理」的作為起漲點

判斷邏輯：
if 有明確題材日期:
    起漲點 = 題材發布日價格
elif 有明顯突破:
    起漲點 = 突破前整理低點
else:
    起漲點 = min(20日低點, 60日低點)

計算漲幅：
漲幅 = (目前價格 - 起漲點) / 起漲點 × 100%

評分公式：
追漲階段分數 = 
  if 漲幅 ≤ 12: 1.5
  elif 漲幅 ≤ 20: 0.5
  elif 漲幅 ≤ 25: 0.2
  else: 0.0


【3.3.2 賠率計算】（5% = 0.5分）
評估：風險報酬比是否划算

計算公式：
賠率 = 預期獲利 / 預期虧損
     = (目標價 - 買進價) / (買進價 - 停損價)

目標價設定：
初升段：買進價 × 1.12（+12%）
主升段：買進價 × 1.10（+10%）
末升段：買進價 × 1.08（+8%）

停損價設定：
初升段：買進價 × 0.945（-5.5%）
主升段：買進價 × 0.94（-6%）
末升段：買進價 × 0.96（-4%）

評分標準：
賠率 ≥ 2.5:1：0.5分 ✅✅✅
賠率 2.0-2.4:1：0.4分 ✅✅
賠率 1.5-1.9:1：0.3分 ✅
賠率 < 1.5:1：0分 ❌

計算公式：
賠率分數 = 
  if 賠率 ≥ 2.5: 0.5
  elif 賠率 ≥ 2.0: 0.4
  elif 賠率 ≥ 1.5: 0.3
  else: 0.0


【追漲安全性總分計算】
總分 = 追漲階段 + 賠率
滿分 = 1.5 + 0.5 = 2.0

評級標準：
= 2.0：滿分 ✅✅✅
1.5-1.9：良好 ✅
1.0-1.4：及格 ⚠️
< 1.0：不及格 ❌

# 完整評分函數

def calculate_score(stock):
    # 1. 題材面評分（4.0分）
    theme_strength = calculate_theme_strength()  # 1.5分
    theme_duration = calculate_theme_duration()  # 1.0分
    theme_freshness = calculate_theme_freshness()  # 1.0分
    theme_authenticity = calculate_theme_authenticity()  # 0.5分
    
    theme_score = (theme_strength + theme_duration + 
                   theme_freshness + theme_authenticity)
    
    # 2. 籌碼面評分（4.0分）
    institutional_buying = calculate_institutional()  # 2.0分
    continuity = calculate_continuity()  # 1.0分
    volume = calculate_volume()  # 0.5分
    ownership = calculate_ownership()  # 0.5分
    
    chip_score = (institutional_buying + continuity + 
                  volume + ownership)
    
    # 3. 追漲安全評分（2.0分）
    stage = calculate_stage()  # 1.5分
    risk_reward = calculate_risk_reward()  # 0.5分
    
    momentum_score = stage + risk_reward
    
    # 4. 總分
    total_score = theme_score + chip_score + momentum_score
    
    return {
        'theme_score': theme_score,
        'chip_score': chip_score,
        'momentum_score': momentum_score,
        'total_score': total_score
    }


# 詳細計算函數

def calculate_theme_strength():
    """題材強度（1.5分）"""
    # 根據影響比例評星
    if impact_ratio >= 70:
        stars = 5
    elif impact_ratio >= 40:
        stars = 4
    elif impact_ratio >= 20:
        stars = 3
    elif impact_ratio >= 10:
        stars = 2
    else:
        stars = 1
    
    return (stars / 5) * 1.5


def calculate_theme_duration():
    """題材持續性（1.0分）"""
    if duration == '長線':  # 3個月以上
        return 1.0
    elif duration == '中線':  # 1-2個月
        return 0.7
    else:  # 短線 3-7天
        return 0.3


def calculate_theme_freshness():
    """題材新鮮度（1.0分）"""
    days = (today - theme_date).days
    
    if days <= 3:
        return 1.0
    elif days <= 7:
        return 0.7
    elif days <= 14:
        return 0.4
    else:
        return 0.0


def calculate_theme_authenticity():
    """真實性（0.5分）"""
    if authenticity == '已確認':
        return 0.5
    elif authenticity == '部分確認':
        return 0.3
    else:
        return 0.0


def calculate_institutional():
    """法人買超（2.0分）"""
    # 加權計算
    weighted_buying = (
        foreign_buying * 0.5 +  # 外資50%
        investment_trust_buying * 0.35 +  # 投信35%
        dealer_buying * 0.15  # 自營15%
    )
    
    # 單日評分（1.0分）
    if weighted_buying >= 500:
        daily_score = 1.0
    elif weighted_buying >= 300:
        daily_score = 0.7
    elif weighted_buying >= 100:
        daily_score = 0.4
    elif weighted_buying > 0:
        daily_score = 0.2
    else:
        daily_score = 0.0
    
    # 累計評分（1.0分）
    if accumulated_buying >= 1000:
        acc_score = 1.0
    elif accumulated_buying >= 500:
        acc_score = 0.7
    elif accumulated_buying > 0:
        acc_score = 0.4
    else:
        acc_score = 0.0
    
    return min(daily_score + acc_score, 2.0)


def calculate_continuity():
    """連續性（1.0分）"""
    continuous_days = count_continuous_buying_days()
    
    if continuous_days >= 5:
        return 1.0
    elif continuous_days >= 3:
        return 0.7
    elif continuous_days == 2:
        return 0.5
    elif continuous_days == 1:
        return 0.3
    else:
        return 0.0


def calculate_volume():
    """成交量（0.5分）"""
    ratio = today_volume / avg_20day_volume
    
    if ratio >= 3:
        return 0.5
    elif ratio >= 2:
        return 0.4
    elif ratio >= 1.5:
        return 0.3
    elif ratio >= 1:
        return 0.1
    else:
        return 0.0


def calculate_ownership():
    """大戶持股（0.5分）"""
    change = this_week_ownership - last_week_ownership
    
    if change >= 3:
        return 0.5
    elif change >= 2:
        return 0.4
    elif change >= 1:
        return 0.3
    elif change > 0:
        return 0.1
    else:
        return 0.0


def calculate_stage():
    """追漲階段（1.5分）"""
    # 找起漲點
    starting_point = find_starting_point()
    
    # 計算漲幅
    gain = (current_price - starting_point) / starting_point * 100
    
    if gain <= 12:
        return 1.5  # 初升段
    elif gain <= 20:
        return 0.5  # 主升段前期
    elif gain <= 25:
        return 0.2  # 主升段後期
    else:
        return 0.0  # 末升段


def find_starting_point():
    """找起漲點"""
    # 優先順序：
    # 1. 題材發布日價格
    # 2. 突破整理低點
    # 3. 近期低點
    
    if theme_date:
        return price_on_theme_date
    elif has_breakout:
        return breakout_base_price
    else:
        return min(low_20day, low_60day)


def calculate_risk_reward():
    """賠率（0.5分）"""
    # 根據階段設定目標與停損
    gain = calculate_current_gain()
    
    if gain <= 12:  # 初升段
        target = current_price * 1.12
        stop_loss = current_price * 0.945
    elif gain <= 25:  # 主升段
        target = current_price * 1.10
        stop_loss = current_price * 0.94
    else:  # 末升段
        target = current_price * 1.08
        stop_loss = current_price * 0.96
    
    # 計算賠率
    profit = target - current_price
    loss = current_price - stop_loss
    ratio = profit / loss if loss > 0 else 0
    
    if ratio >= 2.5:
        return 0.5
    elif ratio >= 2.0:
        return 0.4
    elif ratio >= 1.5:
        return 0.3
    else:
        return 0.0
