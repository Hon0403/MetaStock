# 📋 MetaStock 功能規劃總表

## 🎯 核心目標
根據「外資報告陷阱速查手冊」的策略，將手動檢查流程自動化。

---

## ✅ 已完成 - 資料層 (9個表)
| 資料 | 表名 | 用途 |
|:---|:---|:---|
| 股票清單 | `stocks` | 基礎資料 |
| 每日股價 | `prices` | 技術分析 |
| 三大法人 | `institutional` | 籌碼分析 |
| 融資融券 | `margins` | 籌碼分析 |
| 集保股權 | `shareholders` | 籌碼分析 |
| 月營收 | `revenues` | 基本面 |
| 本益比/殖利率 | `valuations` | 價值分析 |
| 當沖交易 | `daytrades` | 籌碼分析 |
| 除權除息 | `dividends` | 還原股價 |

## 📋 待開發功能

### Phase 1: 外資報告驗證工具 ★★★★★
**對應**：外資報告陷阱速查手冊

| 功能 | 來源資料 | 說明 |
|:---|:---|:---|
| 籌碼驗證 | `institutional` | 查詢近 10 天買賣超 |
| 時間回推法 | `prices` | 回推 30 天股價變化 |
| 漲幅檢查 | `prices` | 計算起漲點漲幅 |
| 持股變化 | `institutional` | 查詢外資持股趨勢 |

### Phase 2: 智慧選股系統 ★★★★☆
| 功能 | 來源資料 | 說明 |
|:---|:---|:---|
| 價值成長股篩選 | `valuations` + `revenues` | PE < 15 且營收成長 |
| 高股息存股 | `valuations` + `dividends` | 殖利率 > 5% 且穩定配息 |
| 籌碼異動偵測 | `institutional` + `margins` | 法人買超 + 融資減 |

### Phase 3: 券商分點搜尋 ★★★★★ (待解決驗證碼)

#### 功能需求
- **輸入**：外資/券商名稱 (如「摩根大通」)
- **輸出**：該券商的資金流向 (買什麼、賣什麼)

#### 必備資料表
| 表名 | 說明 |
|:---|:---|
| `brokers` | 券商代號對照 (1470 = 摩根士丹利) |
| `brokertrades` | 券商每日買賣明細 |

#### 常見外資代號
| 代號 | 名稱 |
|:---|:---|
| 1470 | 台灣摩根士丹利 (Morgan Stanley) |
| 1440 | 美林 (Merrill Lynch) |
| 1480 | 美商高盛 (Goldman Sachs) |
| 1560 | 港商野村 (Nomura) |

#### 阻塞問題
- ❌ 資料來源 (bsr.twse.com.tw) 有**驗證碼保護**
- 解法：OCR 自動辨識 或 付費 API
