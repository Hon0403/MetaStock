# 計畫：市場估值指標與進階分析 (Valuations)

## 1. 目標
整合台灣證交所的 **BWIBBU** (本益比、殖利率、股價淨值比) 資料，讓系統從單純的「看盤」升級為「策略選股」平台。

## 2. 資料庫結構 (Data Stack)

我們目前的資料庫已經具備了以下拼圖，加上 **Valuations** 後將完成最後一塊：

| 資料表 | 內容 | 用途 |
| :--- | :--- | :--- |
| `stocks` | 股票清單 | 基礎代號與名稱 |
| `prices` | 每日股價 | 技術分析 (K線、均線) |
| `institutional` | 三大法人 | 籌碼分析 (洋大人動向) |
| `margins` | 融資融券 | 籌碼分析 (散戶信心) |
| `shareholders` | 集保股權 | 籌碼分析 (大戶 vs 散戶) |
| `revenues` | 月營收 | 基本面 (成長力道) |
| **`valuations` (NEW)** | **估值指標** | **價值評估 (便宜 vs 貴)** |

## 3. 選股策略與資料組合 (Combination Logic)

利用上述資料表的組合，我們可以產出以下強力策略：

### A. 「價值成長股」策略 (Undervalued Growth)
*   **邏輯**：公司賺錢越賺越多 (成長)，但股價還沒反應 (便宜)。
*   **資料組合**：
    *   `revenues` (營收年增率 > 20%)
    *   `valuations` (本益比 PE < 15)
*   **應用場景**：每月營收公布時 (每月 10 號) 掃描一次。

### B. 「高股息存股」策略 (High Yield & Stability)
*   **邏輯**：真正發得出錢，而且專業經理人 (投信) 認證過的好公司。
*   **資料組合**：
    *   `valuations` (殖利率 > 5%)
    *   `institutional` (投信 = 買超)
*   **應用場景**：除權息旺季前 (4-6月) 佈局。

### C. 「破底翻/抄底」策略 (Floor Detection)
*   **邏輯**：股價跌爛 (低於淨值)，這時候散戶因為虧損恐慌殺出 (融資大減)，通常是強力買點。
*   **資料組合**：
    *   `valuations` (股價淨值比 PB < 0.8)
    *   `margins` (融資餘額 = 大減)
*   **應用場景**：大盤崩盤時尋寶。

### D. 「聰明錢估值」 (Smart Money)
*   **邏輯**：大戶 (千張持股) 通常比散戶更懂公司價值。如果 PE 很低且大戶一直在買，代表被低估。
*   **資料組合**：
    *   `valuations` (本益比 PE < 12)
    *   `shareholders` (大戶持股比例 = 上升)
*   **應用場景**：每週五集保數據公布後。

## 4. 未來必做資料 (Future Roadmap)

為了讓分析更精準，完成 Valuations 後，建議補齊以下資料：

### 1. 除權息與股利政策 (Dividends) ★★★★★
*   **用途**：計算「還原權值股價」。
*   **為什麼必做**：如果沒有還原股價，股票除息那天股價跌下來，系統會誤判為「大跌/轉空」，這是嚴重的雜訊。

### 2. 每季財報 EPS (Quarterly Financials) ★★★★☆
*   **用途**：計算「本益成長比 (PEG)」。
*   **為什麼必做**：目前的本益比是「股價 / 過去 EPS」，但股價反應的是未來。我們需要更細的財報數據來預估未來的 EPS。

### 3. 產業分類 (Industry) ★★★☆☆
*   **用途**：資金流向分析。
*   **為什麼必做**：可以知道現在錢是在流向「半導體」還是「航運」。單看一檔股票容易受大盤影響，看產業比較準。

## 5. 進階籌碼資料 (Advanced Roadmap - Stage 2)

如果您想做到「主力級」的監控，以下這兩項是台股最特別的數據：

### 1. 券商分點進出 (Broker Branch) ★★★★★
*   **用途**：抓出「隔日沖」、「地緣主力」、「關鍵分點」。
*   **說明**：這比三大法人更細，可以看到是「凱基台北」還是「虎尾幫」在買。這是台股籌碼分析的終極聖杯。

### 2. 董監事持股/申報轉讓 (Insider Ownership) ★★★★☆
*   **用途**：避雷針 (內部人落跑訊號)。
*   **說明**：春江水暖鴨先知，如果股價漲但老闆一直在賣股票 (申報轉讓)，通常是高點到了。
