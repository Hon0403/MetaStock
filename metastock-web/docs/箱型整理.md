Step 1：箱型整理辨識
text
目的：
找出正在橫盤整理的股票

【1.1 什麼是箱型整理】

定義：
股價在固定區間內上下震盪
高點大約在同一位置（壓力線）
低點大約在同一位置（支撐線）

圖示：
高點 55 ────────────────── 壓力線
         │震盪│震盪│震盪│
低點 50 ────────────────── 支撐線
         ↑    ↑    ↑
       整理  整理  整理

特徵：
✅ 高點至少碰 2-3 次（形成壓力）
✅ 低點至少碰 2-3 次（形成支撐）
✅ 箱子高度：5-10%（太小沒意義，太大不是整理）
✅ 整理時間：1-4 週（10-20 個交易日）
✅ 成交量逐步萎縮（等待方向）


【1.2 箱型辨識條件】

必要條件（全部符合）：
□ 近 10 日漲跌幅在 ±8% 內（橫盤）
□ 最高價與最低價差距 5-10%（箱子高度）
□ 至少震盪 10 天以上（整理時間）
□ 成交量逐步萎縮（量能變小）

加分條件（符合越多越好）：
□ 均線糾結（5、20、60 日線距離 < 5%）
□ 有明顯高低點（至少碰 3 次）
□ 之前有一段漲幅（整理後再起）
□ 股價在均線之上（多頭整理）

def consolidation_filter(stock):
    """整理型態辨識"""
    # 計算近 10 日價格區間
    high_10d = max(stock.high_prices[-10:])
    low_10d = min(stock.low_prices[-10:])
    price_range = (high_10d - low_10d) / low_10d * 100
    
    # 計算漲跌幅
    start_price = stock.close_prices[-10]
    end_price = stock.close_prices[-1]
    change = (end_price - start_price) / start_price * 100
    
    # 計算成交量趨勢
    volume_recent = avg(stock.volumes[-5:])
    volume_before = avg(stock.volumes[-15:-5])
    volume_ratio = volume_recent / volume_before
    
    return (
        price_range >= 5 and price_range <= 10 and  # 箱子高度 5-10%
        abs(change) <= 8 and                        # 整體漲跌 < 8%
        volume_ratio < 0.7                          # 量能萎縮 < 70%
    )

def ma_convergence_filter(stock):

def support_resistance_filter(stock):
    """確認支撐壓力明確"""
    prices = stock.close_prices[-20:]  # 近 20 日
    
    # 找高點（接近最高價的次數）
    high = max(prices)
    high_touches = sum(1 for p in prices if p >= high * 0.98)
    
    # 找低點（接近最低價的次數）
    low = min(prices)
    low_touches = sum(1 for p in prices if p <= low * 1.02)
    
    return (
        high_touches >= 2 and  # 壓力線至少碰 2 次
        low_touches >= 2       # 支撐線至少碰 2 次
    )

def calculate_consolidation_days(stock):
    """計算整理了多久"""
    prices = stock.close_prices[-30:]  # 看近 30 天
    
    # 找出開始整理的日期
    for i in range(len(prices) - 1, -1, -1):
        # 檢查這天之前是否有明顯方向
        before = prices[max(0, i-5):i]
        if has_trend(before):  # 之前有趨勢
            return len(prices) - i
    
    return 0  # 找不到整理起點

def has_trend(prices):
    """判斷是否有趨勢"""
    if len(prices) < 3:
        return False
    
    # 計算斜率
    slope = (prices[-1] - prices) / len(prices)
    
    # 判斷是否為明顯趨勢（絕對值 > 0.5%）
    return abs(slope / prices * 100) > 0.5

目的：

【2.1 什麼是真突破】

定義：
股價帶量突破壓力線
並且收盤站穩在壓力線之上

圖示（真突破）：
         ↗ 收 58 元（站穩）
       ↗
55 ──┴─────────── 壓力線
     突破！
     成交量爆大

vs

圖示（假突破）：
       │ 最高 56（突破）
       ↓ 收 54 元（縮回）❌
55 ────┴────────── 壓力線
       假突破！


【2.2 真突破的 5 大特徵】

特徵 1：量能爆大（最重要！）⭐⭐⭐⭐⭐
標準：
✅ 突破日成交量 ≥ 20 日均量 × 2 倍
✅ 最好 ≥ 3 倍（爆量）

原因：
量是價的先行指標
有量才有人買
沒量的突破是假的

計算：
```python
def check_volume_breakout(stock, breakout_day):
    """檢查突破日量能"""
    volume_today = stock.volume[breakout_day]
    avg_volume_20d = avg(stock.volumes[-20:])
    
    ratio = volume_today / avg_volume_20d
    
    if ratio >= 3:
        return 1.0, "爆量突破 ✅✅✅"
    elif ratio >= 2:
        return 0.7, "量能足夠 ✅✅"
    elif ratio >= 1.5:
        return 0.4, "量能普通 ✅"
    else:
        return 0.0, "量能不足 ❌"
特徵 2：K線強勢（紅K 實體大）⭐⭐⭐⭐
標準：
✅ 收紅 K（收盤 > 開盤）
✅ 紅 K 實體 ≥ 3%（大紅 K）
✅ 上影線很短（< 1%）
✅ 收盤價在當日最高價附近

def check_candle_strength(stock, day):
    """檢查 K 線強度"""
    open_price = stock.open[day]
    close_price = stock.close[day]
    high_price = stock.high[day]
    low_price = stock.low[day]
    
    # 紅 K 實體大小
    body = (close_price - open_price) / open_price * 100
    
    # 上影線長度
    upper_shadow = (high_price - close_price) / close_price * 100
    
    # 收盤位置（在當日振幅的哪裡）
    close_position = (close_price - low_price) / (high_price - low_price)
    
    if body >= 3 and upper_shadow < 1 and close_position > 0.9:
        return 1.0, "強勢紅 K ✅✅✅"
    elif body >= 2 and upper_shadow < 2:
        return 0.6, "標準紅 K ✅"
    else:
        return 0.0, "K 線弱 ❌"
特徵 3：突破時間（早盤突破最好）⭐⭐⭐
標準：
✅ 早盤（09:00-10:00）就突破：最好 ✅✅✅
✅ 午盤（10:00-12:00）突破：可以 ✅
⚠️ 尾盤（13:00-13:30）突破：小心 ⚠️

原因：
早盤突破 = 有買盤
尾盤突破 = 可能是拉抬、誘多

def check_breakout_timing(stock, day):
    """檢查突破時間"""
    # 需要盤中資料，檢查何時突破壓力線
    # 如果沒有盤中資料，可以用以下替代：
    
    open_price = stock.open[day]
    high_price = stock.high[day]
    close_price = stock.close[day]
    resistance = stock.resistance_level  # 壓力線價格
    
    # 如果開盤就高於壓力 = 跳空突破（好）
    if open_price > resistance:
        return 1.0, "跳空突破 ✅✅✅"
    
    # 如果收盤遠高於壓力 = 早盤突破機率高
    elif close_price > resistance * 1.02:
        return 0.8, "強勢突破 ✅✅"
    
    # 如果收盤僅略高於壓力 = 可能尾盤才破
    elif close_price > resistance * 1.01:
        return 0.4, "需觀察 ⚠️"
    
    else:
        return 0.0, "未突破或假突破 ❌"

標準：

def check_ma_alignment(stock, day):
    """檢查均線排列"""
    price = stock.close[day]
    ma5 = stock.moving_average(5, day)
    ma20 = stock.moving_average(20, day)
    ma60 = stock.moving_average(60, day)
    
    # 檢查價格是否突破均線
    above_ma = price > ma5 and price > ma20
    
    # 檢查均線是否向上（比 5 天前高）
    ma5_up = ma5 > stock.moving_average(5, day-5)
    ma20_up = ma20 > stock.moving_average(20, day-5)
    
    # 檢查是否黃金交叉
    golden_cross = ma5 > ma20 and ma20 > ma60
    
    if above_ma and ma5_up and ma20_up and golden_cross:
        return 1.0, "均線完美 ✅✅✅"
    elif above_ma and (ma5_up or ma20_up):
        return 0.6, "均線配合 ✅"
    else:
        return 0.3, "均線不佳 ⚠️"
特徵 5：籌碼配合（法人買超）⭐⭐⭐

✅ 外資當日買超 > 100 張

✅ 內外盤比例健康（外盤 > 50%）

def check_chip_support(stock, day):
    """檢查籌碼配合"""
    foreign_buy = stock.foreign_net_buy[day]
    trust_buy = stock.trust_net_buy[day]
    
    # 內外盤比例
    buy_volume = stock.buy_volume[day]  # 外盤
    sell_volume = stock.sell_volume[day]  # 內盤
    buy_ratio = buy_volume / (buy_volume + sell_volume)
    
    score = 0
    messages = []
    
    if foreign_buy > 100:
        score += 0.4
        messages.append("外資買超 ✅")
    
    if trust_buy > 50:
        score += 0.3
        messages.append("投信買超 ✅")
    
    if buy_ratio > 0.5:
        score += 0.3
        messages.append("買盤強勁 ✅")
    
    return score, " ".join(messages)

def calculate_breakout_score(stock, day):
    """計算突破分數（總分 5.0）"""
    
    # 1. 量能（1.0 分）
    volume_score, volume_msg = check_volume_breakout(stock, day)

    candle_score, candle_msg = check_candle_strength(stock, day)

    return {
        'total_score': total,
        'volume': (volume_score, volume_msg),
        'candle': (candle_score, candle_msg),
        'timing': (timing_score, timing_msg),
        'ma': (ma_score, ma_msg),
        'chip': (chip_score, chip_msg)
    }

def is_false_breakout(stock, day):
    """判斷是否為假突破"""
    
    # 檢查 1：量能不足
    volume_ratio = stock.volume[day] / avg(stock.volumes[-20:])
    if volume_ratio < 1.5:
        return True, "量能不足 ❌"
    
    # 檢查 2：長上影線
    high = stock.high[day]
    close = stock.close[day]
    upper_shadow = (high - close) / close * 100
    if upper_shadow > 3:
        return True, "長上影線 ❌"
    
    # 檢查 3：高開低走
    open_price = stock.open[day]
    if open_price > stock.resistance and close < open_price:
        return True, "高開低走 ❌"
    
    # 檢查 4：隔日確認
    if day < len(stock.close) - 1:
        next_close = stock.close[day + 1]
        if next_close < stock.resistance:
            return True, "隔日跌破 ❌"
    
    return False, "非假突破"

class BoxBreakoutScorer:
    """箱型突破評分系統"""
    
    def calculate_total_score(self, stock, breakout_day):
        """計算總分（滿分 10.0）"""
        
        # 1. 箱型品質（3.0 分）
        box_score = self.score_box_quality(stock)
        
        # 2. 突破品質（5.0 分）
        breakout_score = self.score_breakout_quality(stock, breakout_day)
        
        # 3. 風險控管（2.0 分）
        risk_score = self.score_risk_management(stock, breakout_day)
        
        total = box_score + breakout_score + risk_score
        
        return {
            'total_score': total,
            'box_score': box_score,
            'breakout_score': breakout_score,
            'risk_score': risk_score,
            'grade': self.get_grade(total)
        }
    
    def score_box_quality(self, stock):
        """箱型品質評分（3.0 分）"""
        score = 0
        
        # 1.1 箱子高度（1.0 分）
        box_height = calculate_box_height(stock)
        if 5 <= box_height <= 10:
            score += 1.0
        elif 4 <= box_height < 5 or 10 < box_height <= 12:
            score += 0.6
        else:
            score += 0.2
        
        # 1.2 整理時間（1.0 分）
        consolidation_days = calculate_consolidation_days(stock)
        if 15 <= consolidation_days <= 20:
            score += 1.0
        elif 10 <= consolidation_days < 15 or 20 < consolidation_days <= 30:
            score += 0.6
        else:
            score += 0.3
        
        # 1.3 支撐壓力明確度（1.0 分）
        touches = count_support_resistance_touches(stock)
        if touches >= 3:
            score += 1.0
        elif touches == 2:
            score += 0.6
        else:
            score += 0.3
        
        return score
    
    def score_breakout_quality(self, stock, day):
        """突破品質評分（5.0 分）"""
        score = 0
        
        # 2.1 量能（1.0 分）
        volume_ratio = stock.volume[day] / avg(stock.volumes[-20:])
        if volume_ratio >= 3:
            score += 1.0
        elif volume_ratio >= 2:
            score += 0.7
        elif volume_ratio >= 1.5:
            score += 0.4
        
        # 2.2 K線強度（1.0 分）
        candle_strength = calculate_candle_strength(stock, day)
        score += candle_strength
        
        # 2.3 突破時間（1.0 分）
        timing_score = score_breakout_timing(stock, day)
        score += timing_score
        
        # 2.4 均線配合（1.0 分）
        ma_score = score_ma_alignment(stock, day)
        score += ma_score
        
        # 2.5 籌碼配合（1.0 分）
        chip_score = score_chip_support(stock, day)
        score += chip_score
        
        return score
    
    def score_risk_management(self, stock, day):
        """風險控管評分（2.0 分）"""
        score = 0
        
        # 3.1 賠率（1.0 分）
        risk_reward = calculate_risk_reward_ratio(stock, day)
        if risk_reward >= 2.5:
            score += 1.0
        elif risk_reward >= 2.0:
            score += 0.7
        elif risk_reward >= 1.5:
            score += 0.4
        
        # 3.2 安全檢查（1.0 分）
        safety_checks = perform_safety_checks(stock, day)
        passed = sum(safety_checks.values())
        total = len(safety_checks)
        score += (passed / total) * 1.0
